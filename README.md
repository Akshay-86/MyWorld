# 🌍 MyWorld

> A stylized 3D nature simulation built with Three.js — **Work In Progress**

![Status](https://img.shields.io/badge/status-WIP-yellow)

## About

An interactive, explorable 3D world featuring rolling hills, dense forests, a tranquil pond, and a small village connected by dirt paths. Built with stylized low-poly assets from the [Quaternius](https://quaternius.com/) and [Kenney](https://kenney.nl/) nature kits.

## Controls

| Input | Action |
|-------|--------|
| `W` / `↑` | Move forward |
| `S` / `↓` | Move backward |
| `A` / `←` | Strafe left |
| `D` / `→` | Strafe right |
| `Scroll` | Zoom in/out |
| `Left-click drag` | Rotate view |
| `Shift + drag` | Pan |

## Tech Stack

- **Three.js** — 3D rendering
- **Vite** — Dev server & bundler
- **GLTF models** — Quaternius Ultimate Stylized Nature + Kenney Nature Kit

## Run Locally

```bash
npm install
npm run dev
```
